## Definición legal (Art. 2)

Barrera para control vehicular: Dispositivo dotado de punzones pincha llantas para uso en retenes y puesto de control de las fuerzas militares, la Policía Nacional, las autoridades de tránsito y transporte.

## Explicación ciudadana

Barrera para control vehicular, según el Código Nacional de Tránsito (Ley 769 de 2002), se define como: dispositivo dotado de punzones pincha llantas para uso en retenes y puesto de control de las fuerzas militares, la Policía Nacional, las autoridades de tránsito y transporte.
